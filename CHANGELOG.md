# Changelog

All notable changes to this project will be documented in this file.

## [0.2.8.0] - 2026-04-12

### Fixed
- 業務管理（月次）の従業員名テキスト検索でフィルタリングした状態で業務割当を保存すると、フィルタリングされた従業員が画面から消えるバグを修正。検索テキストをURLパラメータ `monthlyEmployeeSearch` に保存することで、保存後のデータ更新時もフィルタリング状態が維持されるよう変更
- 業務割当フォームに `onSuccess` コールバックを追加し、保存成功後に `router.refresh()` を呼び出してサーバーデータを確実に再取得するよう改善
- タイマーリーク修正: 従業員名検索のデバウンスタイマーがコンポーネントアンマウント時にクリーンアップされるよう対応

### Added
- 従業員名検索 URLパラメータ初期化のリグレッションテストを追加（`monthlyEmployeeSearch` パラメータによる検索ボックスの初期化）

## [0.2.7.0] - 2026-04-12

### Fixed
- 今日の夜勤シフト（例: 22:00〜08:00）が翌朝の「対応可能人数」に誤カウントされるバグを修正。夜勤者は開始時刻（22:00）以降のみ在勤扱いとなり、深夜0時〜終了時刻の範囲は前日からの引き継ぎ処理が正しく担当するよう整理
- 退職日を削除して在職者に戻した後、業務管理画面に即時反映されないキャッシュ問題を修正。従業員データ更新時に業務管理画面のキャッシュも再検証するよう変更
- ダッシュボードの夜勤・業務担当取得クエリに退職者除外フィルターを追加。退職済み従業員がリアルタイム集計に混入しないよう対応

### Added
- 夜勤シフトの誤カウント修正に関するユニットテストを追加（`isWorkerPresent` の `forTodayShift` フラグ、`calculateFilteredCapacity`・`extractFilterOptions` の夜勤判定）

## [0.2.6.0] - 2026-04-11

### Added
- 業務管理（月次）のセルを上段（シフト）/下段（業務）に分割。上段クリックでシフト詳細ダイアログまたはシフト新規作成フォームを表示
- 業務管理画面からシフトの確認・作成・編集が可能に。シフト表への画面遷移が不要に
- シフト操作後の業務管理画面の即時反映（revalidatePath追加）

## [0.2.5.0] - 2026-04-11

### Added
- グループマスタに「省略名」カラムを追加。ダッシュボードの本日の業務エリアで省略名を優先表示
- グループ作成・編集フォームに省略名フィールドを追加（10文字以内）
- グループ一覧テーブルに省略名カラムを追加

### Changed
- 業務割当作成・編集ダイアログのフィールド順序を変更。タイトルを一番上に配置
- 本日の業務エリアでタイトルを表示（備考から変更）
- グループ省略名のバリデーションにホワイトスペーストリムを追加

## [0.2.4.0] - 2026-04-11

### Added
- 業務割当に「タイトル」カラムを追加。業務種別とは別に具体的な用件（例: A社訪問）を記録可能に
- 業務種別マスタに「デフォルトタイトル」を追加。業務割当作成時にタイトルが自動補完される
- 月次カレンダーのセル表示を色ドットからタイトルテキスト表示に変更。クリックなしで業務内容を把握可能に
- セル内のタイトルが長い場合はellipsis処理、3件以上の場合は「他N件」のオーバーフロー表示

### Changed
- 月次カレンダーセルの表示方式を「色ドット（最大6個）」から「色付きタイトルテキスト（最大2件+他N件）」に変更
- Dialogの業務一覧にタイトルを表示（業務種別名: タイトル）
- 業務割当フォームのDutyType変更時、タイトルの自動補完は新規作成時のみに限定（編集時は上書きしない）

## [0.2.3.0] - 2026-04-11

### Removed
- 業務種別マスタの「業務コード（code）」カラムを完全廃止。識別はID、表示はnameに統一
- 業務種別フォームのコード入力フィールドを削除
- 日次ビューのコードバッジ表示を廃止し、業務名テキスト表示に変更
- 業務コードの重複チェック（P2002エラーハンドリング）を削除

### Changed
- 月次カレンダーの業務種別集計キーをcodeからIDに変更
- 月次カレンダーの業務種別ソートをcodeアルファベット順からsortOrder順に変更
- 日次ビューのフィルター選択肢ソートをname日本語順に変更

## [0.2.2.1] - 2026-04-10

### Changed
- 業務管理（月次）画面のセルクリック時の業務一覧表示をPopoverから画面中央のモーダルDialogに変更（シフト管理画面と統一）

## [0.2.2.0] - 2026-04-10

### Added
- 月次カレンダーのセルクリックで業務割当の詳細Popoverを表示（業務一覧・編集・削除・新規追加）
- 業務種別にデフォルト値（開始時刻・終了時刻・備考）を設定可能に
- 月次カレンダーのセル内にシフトコードバッジを表示
- 業務種別Server Actionテスト、バリデーションテスト、DBクエリのnoteフィールドテストを追加

### Changed
- 業務割当フォームを制御コンポーネントに変更し、業務種別選択時にデフォルト値を自動入力
- 日付比較クエリを文字列::date型キャストに変更し、タイムゾーン依存を排除

## [0.2.1.3] - 2026-04-08

### Fixed
- 業務管理（月次）画面で未ログインユーザーがセルクリック時に業務割当ダイアログが表示される問題を修正

### Added
- コンポーネントテスト基盤を導入（@testing-library/react + happy-dom）
- 月次カレンダー認証ガードのコンポーネントテストを追加

### Changed
- 月次カレンダーの従業員名セルからグループ名表示を削除し、レイアウトを簡素化

## [0.2.1.2] - 2026-04-06

### Fixed
- シフト更新・削除後にダッシュボード（`/`）が自動更新されない問題を修正。`updateShift` および `deleteShift` に `revalidatePath("/")` を追加

## [0.2.1.1] - 2026-04-06

### Changed
- 「夜勤は除く」チェックボックスの状態をURLパラメータとlocalStorageで永続化し、ページリロード後も選択状態を維持するように改善

## [0.2.1.0] - 2026-04-06

### Added
- タイムラインヒートマップの夜勤対応: 「夜勤は除く」チェックOFF時に0:00-24:00の全時間帯表示に切替
- 前日夜勤（日跨ぎシフト）の翌日分を0:00-終了時刻としてタイムラインに表示
- 同一従業員の前日夜勤と当日シフトを1行にマージして表示
- タイムライン内の名前検索（夜勤のみの従業員も検索可能）
- タブ連動の人数表示（タイムラインタブではマージ後の人数、リストタブでは今日のシフト数）

### Changed
- 月次カレンダーのセル表示をバッジ（テキスト付き）から色付きドットに変更し、セル幅内に収まるように改善

## [0.2.0.0] - 2026-04-06

### Added
- 業務管理の日次ビュー: 従業員・グループ・業務種別・控除フィルター付きテーブル、カーソルベースの無限スクロールページネーション
- 業務管理の月次カレンダービュー: 従業員×日グリッドに業務種別バッジ表示、グループフィルター、業務種別別月間集計行
- 日次タイムラインバーチャート: 従業員ごとの業務を時間軸で視覚化、重なり業務のレーン分割表示
- ビュー切り替え（月次⇄日次）: URLパラメータ連動、カレンダーセルクリックで日次ビューへドリルダウン
- フィルタープリセット保存: よく使うフィルター条件をlocalStorageに保存・復元（日次/月次別、最大10件）
- 日次ビューのクイック編集: 認証済みユーザーは行クリックで編集ダイアログを開ける
- フィルターコンポーネントの共通化: shifts/配下のフィルターをcomponents/common/filters/に移動し再利用可能に

### Changed
- 業務管理ページをServer Component + Client Componentの分離構成にリファクタリング
- DBクエリ関数のSQLインジェクション対策: パラメータ化クエリとUUIDバリデーションを適用

### Previous releases included in this version
- v0.1.4.0: 業務割当のシフト整合性バリデーション（シフト時間外・休日ブロック、深夜跨ぎ対応）
- v0.1.3.0: タイムラインヒートマップのフィルター追加、ダッシュボードレイアウト改良
- v0.1.2.0: タイムラインヒートマップビュー追加
- v0.1.1.0: ロール割り当てCSVインポート機能
