## 2. ビジネスモデル提案

### 2.1 推奨ビジネス: 仕訳チェック＆修正提案Agent

**ターゲット顧客**:
- 会計事務所（全国約3万件）
- 中小企業の経理部門
- 個人事業主・フリーランス

**提供価値**:
- CSVアップロード → 仕訳の異常検知（勘定科目ミス、消費税率エラー等）
- 修正提案を自動生成
- カスタマーサポート経験から「よくあるミス」を熟知している強み

**なぜ勝てるか**:
1. あなたは「サポートで受けた質問」=「顧客の痛み」を知っている
2. 競合は汎用AI。あなたは会計特化で精度が出せる
3. 技術力があるので、迅速にMVPを構築できる

## 3. Agent Teams 構成

### 3.1 システムアーキテクチャ

```
┌─────────────────────────────────────────────────────────┐
│                  仕訳チェックAgent Team                   │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  [入力Agent]          [分析Agent]         [出力Agent]    │
│  ・CSV/Excel受付      ・勘定科目検証      ・レポート生成  │
│  ・フォーマット正規化  ・消費税率チェック  ・修正案提示    │
│                       ・異常値検出                       │
│                                                         │
│  ────────────────────────────────────────────────────  │
│                  [知識ベース]                            │
│  ・勘定科目マスタ  ・税率表  ・よくあるミスパターン        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 3.2 技術スタック

| レイヤー | 技術 | 理由 |
|----------|------|------|
| AIエンジン | OpenAI API / Claude API | 高精度な分析 |
| データ処理 | Python (pandas) | 柔軟なCSV処理 |
| フロントエンド | Next.js + React | あなたの得意領域 |
| データベース | PostgreSQL / Supabase | 信頼性と拡張性 |
| 決済 | Stripe | サブスク課金に最適 |
| ホスティング | Vercel | Next.jsと相性抜群 |

### 3.3 Agent詳細設計

#### 入力Agent
```
役割: データ受け取りと前処理
入力: CSV, Excel, 手入力
出力: 正規化された仕訳データJSON

処理内容:
1. ファイル形式の判定
2. 列マッピング（日付、勘定科目、金額等）
3. データクレンジング
4. 分析Agentへの引き渡し
```

#### 分析Agent
```
役割: 仕訳内容の検証と異常検出
入力: 正規化された仕訳データ
出力: 検証結果と修正提案

チェック項目:
1. 勘定科目の妥当性
2. 消費税率の整合性（10%, 8%, 非課税）
3. 借方・貸方の一致
4. 異常金額の検出
5. 日付の妥当性
6. 重複仕訳の検出
```

#### 出力Agent
```
役割: レポート生成とユーザーへの提示
入力: 検証結果
出力: 人間が読みやすいレポート

出力形式:
1. サマリー（OK/警告/エラーの件数）
2. 詳細一覧（問題箇所と修正案）
3. 修正済みCSVのダウンロード
```

## 付録: よくある仕訳ミスパターン（実装候補）

1. **勘定科目の誤り**
   - 消耗品費 ↔ 備品（10万円基準）
   - 交際費 ↔ 会議費
   - 旅費交通費 ↔ 通勤手当

2. **消費税率の誤り**
   - 軽減税率対象（8%）の見落とし
   - 非課税取引の課税処理
   - 免税事業者への課税仕入処理

3. **金額の誤り**
   - 桁間違い
   - 税込/税抜の混在
   - 借方・貸方の不一致

4. **日付の誤り**
   - 月またぎの計上ミス
   - 決算日超えの処理

5. **重複仕訳**
   - 同一取引の二重計上
   - 訂正仕訳の漏れ
